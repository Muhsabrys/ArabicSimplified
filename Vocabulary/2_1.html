Arabic Self-Study Studio

A compact, self-contained vocabulary and practice app for Arabic learners. It includes searchable vocabulary, flashcards, exercise checking, matching practice, progress tracking, and JSON import/export for future lessons.

Study overview

Use the vocabulary section for controlled review, flashcards for active recall, and practice for sentence-level production. Progress is stored locally in this browser only.

Recommended sequence

1. Read the vocabulary cards. 2. Review flashcards. 3. Complete practice exercises. 4. Use matching for quick recall.

Arabic input checking

The checker accepts exact answers and also uses a forgiving Arabic comparison that ignores short vowels and tatweel.

Teacher controls

The Data tab lets you paste or import JSON, export the current dataset, and reset saved progress.
